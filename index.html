Word Runner é um jogo de corrida de palavras, onde o jogador deve digitar
as palavras que aparecem na tela para que o carro ande mais rápido!
Quanto mais rápido o carro andar, mais pontos o jogador ganha!
Se o jogador errar uma palavra, o carro perde velocidade!
Use seus pontos para subir em rankings das ligas!
